## Setup Local

1. **Install dependencies**:
   ```bash
   npm install

   ```

   
   Get your key from: [OpenAI API Keys](https://platform.openai.com/api-keys)

3. **Run dev server**:
   ```bash
   npm run dev
   ```
